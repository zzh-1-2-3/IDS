## 系统简介

IDS-Web是一个基于深度学习的网络入侵检测系统，集成了实时流量捕获、智能威胁检测、自适应响应策略等核心功能。系统采用CNN神经网络模型，能够有效识别7种常见的网络攻击类型。

### 设计理念
- **实时性**：秒级流量捕获与检测
- **智能化**：基于深度学习的自动威胁识别
- **自适应**：根据攻击类型自动匹配响应策略
- **可视化**：直观的仪表盘和图表展示
- **可扩展**：支持多种检测算法和策略配置

### 应用场景
- 企业网络安全监控
- 数据中心流量分析
- 网络异常行为检测
- 安全事件响应

## 核心功能

### 1. 实时流量监控
- **多网卡支持**：可同时监控多个网络接口
- **流量捕获**：基于Scapy的高性能数据包捕获
- **特征提取**：自动提取26维流量特征
- **实时展示**：流量趋势、协议分布、IP统计

### 2. 深度学习检测
- **CNN模型**：卷积神经网络，提取流量特征的空间模式
- **两阶段分类**：先区分正常/异常，再细分类攻击类型
- **在线训练**：支持实时数据训练新模型
- **模型评估**：准确率、召回率、F1分数、AUC-ROC

### 3. 威胁评估
- **多维度评分**：综合攻击类型、置信度、流量特征
- **威胁等级**：无危害、低危、中危、高危四级分类
- **权重计算**：动态调整各因素权重
- **历史追踪**：记录威胁变化趋势

### 4. 自适应响应
- **策略匹配**：根据攻击类型和威胁级别自动匹配策略
- **防火墙联动**：自动执行封禁/限流命令
- **白名单机制**：保护可信IP不受误报影响
- **策略管理**：支持自定义策略和自适应策略

### 5. 可视化管理
- **实时仪表盘**：流量统计、威胁概览、策略执行情况
- **图表分析**：流量趋势、攻击分布、协议统计
- **日志查询**：详细的检测日志、策略执行日志
- **模型管理**：训练历史、性能对比、模型切换

## 技术架构

### 前端技术栈
- **框架**：原生HTML5 + CSS3 + JavaScript ES6+
- **图表库**：Chart.js 4.x
- **设计模式**：单页应用（SPA）
- **响应式布局**：支持桌面和移动设备

### 后端技术栈
- **Web框架**：FastAPI（Python 3.8+）
- **数据库**：MySQL 5.7+ + SQLAlchemy ORM
- **认证**：JWT Token + bcrypt密码加密
- **异步处理**：支持高并发流量捕获

### 深度学习技术栈
- **框架**：PyTorch 1.8+
- **模型架构**：CNN（卷积神经网络）
- **数据处理**：NumPy、Pandas、Scikit-learn
- **可视化**：Matplotlib、Seaborn

### 网络安全技术
- **流量捕获**：Scapy（支持TCP/UDP/ICMP）
- **防火墙联动**：Windows netsh / Linux iptables
- **协议分析**：深度包检测（DPI）
- **威胁情报**：攻击特征库、模式匹配

## 使用指南

### 快速开始

#### 1. 系统登录
1. 打开浏览器访问系统地址
2. 使用默认账号登录（admin/admin123）
3. 首次登录后修改密码

#### 2. 配置网卡
1. 进入"实时流量"页面
2. 选择要监控的网络接口
3. 设置更新间隔（建议1-10秒）
4. 点击"开始捕获"

#### 3. 查看检测结果
1. 进入"威胁流量"页面
2. 查看实时检测到的威胁
3. 点击"查看"查看详细信息
4. 查看响应策略执行情况

#### 4. 管理策略
1. 进入"响应策略"页面
2. 查看自适应策略列表
3. 创建自定义策略
4. 启用/禁用策略
5. 查看正在执行的策略

#### 5. 训练模型
1. 进入"模型管理"页面
2. 选择数据集（CICIDS2017）
3. 选择模型类型（CNN）
4. 设置训练参数
5. 点击"开始训练"
6. 查看训练进度和结果

### 高级功能

#### 实时流量筛选
- 支持按源IP、目的IP、端口、协议、攻击类型筛选
- 支持分页查询，每页显示10-50条记录
- 支持时间范围筛选

#### 威胁流量分析
- 查看威胁等级分布
- 分析攻击类型趋势
- 统计高危IP地址
- 导出检测报告

#### 策略配置
- 自定义策略：手动配置IP、端口、协议、动作
- 自适应策略：基于攻击类型和威胁级别自动匹配
- 白名单管理：添加可信IP，避免误报
- 策略执行：手动/自动执行，支持取消操作

## 模块说明

### 流量捕获模块（capture_service.py）
**功能**：实时捕获网络流量数据包

**核心类**：
- `Flow`：流量流类，管理单个连接的完整生命周期
- `FlowManager`：流量管理器，维护所有活跃流量流
- `CaptureService`：捕获服务，提供统一的捕获接口

**关键方法**：
- `start_capture()`：启动流量捕获
- `stop_capture()`：停止捕获并返回已完成的流量
- `get_flows()`：获取所有流量流
- `process_packet()`：处理单个数据包

### 模型服务模块（model_service.py）
**功能**：管理深度学习模型的加载、预测和训练

**核心类**：
- `ModelService`：模型服务单例，管理当前激活的模型

**关键方法**：
- `predict()`：使用当前模型进行预测
- `load_model()`：加载指定模型文件
- `train_model()`：在线训练新模型
- `evaluate_model()`：评估模型性能

### 检测服务模块（detection_service.py）
**功能**：流量检测和威胁评估

**核心方法**：
- `detect_traffic()`：检测单条流量
- `batch_detect()`：批量检测流量
- `calculate_threat_level()`：计算威胁等级

### 策略服务模块（strategy_service.py）
**功能**：策略匹配和执行

**核心方法**：
- `find_matching_strategy()`：查找匹配的自适应策略
- `execute_adaptive_strategy()`：执行自适应策略
- `execute_strategy()`：执行自定义策略
- `cancel_strategy()`：取消执行的策略

### 威胁评分模块（threat_scorer.py）
**功能**：多维度威胁评估

**评分因素**：
- 攻击类型权重
- 置信度分数
- 流量特征异常度
- 历史威胁记录

## 算法原理

### CNN模型
**原理**：卷积神经网络，提取流量特征的空间模式

**优势**：
- 自动特征提取
- 平移不变性
- 并行计算能力强

**应用**：适合处理流量特征矩阵

### 两阶段分类
**第一阶段**：二分类（正常 vs 攻击）
- 专注于区分正常和异常流量
- 避免多类别干扰

**第二阶段**：多分类（攻击类型细分类）
- 针对攻击样本进行细分类
- 提供更详细的威胁信息

**优势**：
- 降低类别不平衡影响
- 提升整体准确率
- 便于策略匹配
